<h2>NewsMind</h2>

사용자의 관심 국가/카테고리/키워드를 저장해서 맞춤 뉴스 피드를
<br/>
Azure 번역 API로 번역해서 AI로 요약/키워드/분석까지 해주는
<br/>
카카오 로그인 기반 개인화 뉴스 서비스.

기사 API : NewsData.io
<br/>
번역 API : Microsoft Azure Translator
<br/>
로그인 API : 카카오 간편로그인
<br/>
Open AI API Model : gpt-4o-mini
<br/>
<br/>
서버 : AWS EC2
<br/>
DB : MySQL
